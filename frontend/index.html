Frequently Asked.

Does Yaper store my messages? +
No. Yaper operates on a session-persistence model. Messages are held in-memory for recent history (up to 50 messages) and are cleared once a room becomes inactive for an extended period.
Can I use Yaper on my phone? +
Absolutely. Yaper is built with a mobile-first philosophy, ensuring the landing page and chat interface are perfectly responsive on any screen size.
How do the mentions work? +
Simply type the name of the user you want to tag followed by three underscores, like alex___. The system handles the rest.
Is it free to use? +
Yes, Yaper is currently free to use with all features enabled including private rooms and custom user limits.
